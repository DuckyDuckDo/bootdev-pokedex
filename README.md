# bootdev-pokedex
Pokedex CLI tool built with GO to help learn Go language as well as HTTP Client within go. 

Key Learnings:
- Set up command line REPL application that awaits user inputs and processes commands and arguments
- Learn to map user input to proper callback functions
- Hook up the callback functions to an external API which provides user and REPL additional information and logic
- Hooked up a caching layer to improve API performance for commonly used endpoints along with ensuring concurrency in performance
- Only dealt with GET requests
